### 用户使用

##### 后台管理

1,使用管理员添加的账号，登录后台
2,令牌管理里面，添加一个key,并且启用
![alt text](imgs/image.png)
3,模型市场去订阅对应的模型
![alt text](imgs/image.png)

##### 接口调用

```
所有调用,header里面加上下面的认证key
{
    "Authorization":"上面管理后台复制的key"
}
```

###### 1.获取订阅模型

